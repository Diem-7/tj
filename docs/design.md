# Design v1

## Direction

Style:

> Premium Trading Cockpit, Dark + Glassmorphism

The provided image is a design reference only.
Its content must not be copied.

The app should use:

- dark blue base
- glass surfaces
- cards
- clear structure
- charts
- large numbers
- calm cockpit feeling

## Functional Rule

Every component has a purpose.
No decoration without function.

UI core sentence:

> Klarheit durch Visualisierung

## Dashboard

The dashboard is a visual trading cockpit.

Rules:

- no tables
- no data deserts
- clear cards
- readable metrics
- charts where they clarify data

Dashboard answers:

- Where do I stand?
- What works?
- What does not?

## Lists

Trade lists use cards, not tables.

For many trades:

- compact trade cards
- list cards
- strong visual hierarchy

## Language

- UI text is German.
- Code and database names are English.

Example:

- internal: `net_pnl`
- UI: `Netto PnL`

## Color Tokens

The direction is approved.
Exact color tokens are not approved yet and must be defined before UI polishing.
